EMBER

Invite to your server

"EMBER" is an interactive chatbot for Discord servers that offers features such as music, trivia, events, quotes, polls, translation, games, reminders, weather forecasts, and jokes. Users can access the bot using the "!dfb" command and enjoy a fun and engaging experience."

EMBER Features

Feature Command Description
Music player $play Play a song or playlist in a voice channel
Trivia game $trivia Start a trivia game with questions on various topics
Quote database $quote Retrieve a random quote from previous conversations within the server
Polls $poll Create and vote on polls within the server
Translation $translate Translate messages between languages
Games $game Play various games such as chess
Reminders $remind Set reminders and receive direct messages at the specified time
Weather forecasts $weather Retrieve weather forecasts for a specific location
Jokes $joke Retrieve a random joke from a joke API
Sing $sing It will play your favourite song
Delete $perge Delete messages
